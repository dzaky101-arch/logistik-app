Panduan Pemasangan Google Apps Script

Ikuti panduan berikut agar database Google Sheets Anda terhubung dengan Scanner ini.

1 Buat Google Spreadsheet Baru
  • Buka Google Drive dan buat spreadsheet baru.
  • Salin (copy) seluruh link spreadsheet tersebut dari address bar browser.
  • Tempelkan (paste) link tersebut ke dalam input **Google Sheet URL / ID** di panel pengaturan kiri.
2 Buka Apps Script & Salin Kode
  • Di spreadsheet Anda, klik menu **Extensions (Ekstensi)** > **Apps Script**.
  • Buka tab **2. Salin Kode Google Apps Script** di atas, lalu klik tombol **Salin Kode** (ID Spreadsheet Anda sudah otomatis dimasukkan ke kode).
  • Hapus semua baris kode bawaan di editor Apps Script Anda, lalu tempelkan (paste) kode tersebut.
  • Klik tombol **Save** (ikon disket).
3 Deploy sebagai Aplikasi Web (Web App)
  • Klik tombol **Deploy (Terapkan)** di kanan atas Apps Script > **New deployment (Penerapan baru)**.
  • Pilih jenis **Web app (Aplikasi web)**.
  • Atur konfigurasi:
    • Execute as: Me (Email Anda)
    • Who has access: Anyone (Siapa saja) ⚠️ *Wajib agar Scanner dapat menulis data.*
  • Klik **Deploy**. Otorisasi akun Google Anda (klik *Advanced/Lanjutan* > *Go to project (unsafe)* > *Allow*).
  • Salin **Web app URL** yang muncul (yang berakhiran `/exec`).
  • Tempelkan ke kolom **Google Web App URL (Deployment URL)** di panel kiri.
4 Uji Coba Otorisasi & Mulai Scan
  • Di dalam editor Apps Script Anda, pilih fungsi **`doGet`** di dropdown menu atas, lalu klik **Run (Jalankan)** untuk menyetujui izin Google Sheets di akun Anda.
  • Kembali ke Scanner, pastikan cursor berkedip di kolom input pindaian besar.
  • Ketik nomor resi tes atau tembak barcode, lalu tekan enter. Data akan langsung terkirim secara otomatis!
