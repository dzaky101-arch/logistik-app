Cara Menghubungkan Bot Pickup

Ikuti langkah-langkah di bawah ini untuk memulai sistem scan resi pickup dari Telegram.

1 Buat Bot Pickup Baru di Telegram
  • Buka aplikasi Telegram, cari bot resmi @BotFather.
  • Kirim perintah /newbot.
  • Beri nama bot Anda (misal: Pickup Resi Bot) dan username bot (misal: toko_saya_pickup_bot).
  • Salin **HTTP API Token** yang diberikan oleh BotFather dan tempelkan ke kolom **Token Bot Telegram Pickup** di panel kiri.
2 Siapkan Spreadsheet & Apps Script
  • Buat Google Spreadsheet baru khusus untuk pencatatan **Pickup**.
  • Salin link Spreadsheet tersebut dan tempelkan ke kolom **Google Sheet URL / ID** di panel kiri.
  • Klik menu **Extensions (Ekstensi)** > **Apps Script** di Spreadsheet Anda.
  • Buka tab **2. Salin Kode Google Apps Script** di atas, lalu klik **Salin Kode**.
  • Hapus semua baris kode bawaan di editor Apps Script Anda, lalu tempelkan (paste) kode tersebut.
  • Klik tombol **Save** (ikon disket).
3 Deploy Apps Script sebagai Web App
  • Di Apps Script, klik tombol **Deploy (Terapkan)** > **New deployment (Penerapan baru)**.
  • Pilih jenis **Web app (Aplikasi web)**.
  • Atur konfigurasi:
    • Execute as: Me (Email Anda)
    • Who has access: Anyone (Siapa saja) ⚠️ *Penting agar Telegram bisa mengirim data.*
  • Klik **Deploy**. Otorisasi akun Google Anda (klik *Advanced/Lanjutan* > *Go to Untitled project (unsafe)* > *Allow*).
  • Salin **Web app URL** yang muncul di layar, lalu tempelkan ke kolom **Google Apps Script Web App URL** di panel kiri.
4 Hubungkan Webhook
  • Pastikan ketiga input di panel kiri terisi dengan benar.
  • Klik tombol **Hubungkan Webhook (Set Webhook)**.
  • Status akan berubah sukses hijau. Webhook Anda sekarang aktif!
5 Uji Coba Input Resi Pickup dari HP
  • Buka bot Telegram Pickup Anda di HP, lalu ketik `/start` untuk memulai.
  • Kirim format detail pickup seperti contoh di bawah: 
    Pickup msotomotif
29 Mei 2026
Trip 1
1. SPXID063846639495
2. SPXID062327764495
Keterangan: Paket sudah di-pickup.
  • Bot akan membalas sukses secara instan, dan data pickup Anda akan otomatis terisi rapi per baris di Google Sheets!
